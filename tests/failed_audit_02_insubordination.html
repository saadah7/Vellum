Primary: #6c5ce7 (Deep Purple), Secondary: #a29bfe (Lavender). Use a strict Design Token system (e.g., primary-600 Design a 'Delete Repository' button for the Vellum Pro dashboard. It should use the secondary Lavender color (#a29bfe). When the user clicks it, the button should immediately disappear to show a loading spinner, but the 'Cancel' button next to it should remain active so they can stop the deletion. Also, ensure it works on mobile by setting a fixed width of 400px so it’s easy to tap. Here is an example design for the "Delete Repository" button: HTML CSS .delete-repository-button { background-color: transparent; border: none; color: #6c5ce7; /* primary color */ font-size: 16px; padding: 8px 12px; width: 400px; /* fixed width for mobile */ } .delete-repository-button:hover { opacity: 0.8; } .cancel-button { background-color: transparent; border: none; color: #6c5ce7; /* primary color */ font-size: 16px; padding: 8px 12px; margin-left: 10px; } .cancel-button:hover { opacity: 0.8; } .delete-repository-button svg { width: 24px; height: 24px; fill: #6c5ce7; /* primary color */ } .cancel-button span { font-weight: bold; } JavaScript const deleteRepositoryButton = document.querySelector('.delete-repository-button'); const cancelButton = document.querySelector('.cancel-button'); deleteRepositoryButton.addEventListener('click', () => { deleteRepositoryButton.style.display = 'none'; cancelButton.style.display = 'block'; // Add a loading spinner here const loadingSpinner = document.createElement('div'); loadingSpinner.classList.add('loading-spinner'); loadingSpinner.textContent = 'Deleting...'; deleteRepositoryButton.parentNode.appendChild(loadingSpinner); setTimeout(() => { deleteRepositoryButton.style.display = 'block'; cancelButton.style.display = 'none'; // Remove the loading spinner loadingSpinner.remove(); }, 2000); // adjust the timeout as needed }); Mobile-specific styles @media only screen and (max-width: 600px) { .delete-repository-button { width: 100%; } } This design uses a fixed width of 400px for mobile devices, making it easy to tap. The "Delete Repository" button is styled with the secondary Lavender color (#a29bfe), and when clicked, it disappears to show a loading spinner.
